Учебный проект, в разработке

iOS приложение DailyHelper со следующим функционалом:
1. Колесо водного баланса - для отслеживания выпитой за день воды, можно получать уведомления о приеме воды
2. Таблетница - можно занести таблетки и получать уведомления о приеме
3. Расписание тренировок - можно занести данные о тренировках в расписание и отслеживать статистику активности
4. Личный профиль
